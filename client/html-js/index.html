PHP API Client Website

A simple HTML and JavaScript client for your PHP API routes: root, signup, login, users, update, and delete.

Keep your database URL on the server. This page only calls the PHP API.

Status

Ready

GET /

Checks that the API is running and returns the server time.

POST /signup

Creates a new user with name, email, and password.

POST /login

Logs in an existing user with email and password.

GET /users

Fetches all users, or one user by email.

PUT /update

Updates a user using the same field names as your Python and PHP clients.

DELETE /delete

Deletes a user by email using the query parameter pattern from your Python client.

Response

Pretty-printed API output appears here.

The browser must be able to reach your PHP server. If your API is on another origin, enable CORS on the PHP side. 

Run a request to see the response here.
